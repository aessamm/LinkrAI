# 01 — Linkrai Product Requirements

## 1. Product Brief

**Linkrai** is a mobile-first AI link memory app for users who save many online resources but struggle to find them later.

Users can save URLs from social media, websites, articles, videos, tools, newsletters, and other sources. Linkrai analyzes each link, generates a summary, auto-categorizes it, extracts tags/keywords, and makes the saved library searchable by keyword and semantic meaning.

## 2. One-Line Pitch

**Linkrai is an AI-powered app that saves, summarizes, organizes, and helps users rediscover links by meaning.**

Arabic:

## 3. Core Problem

Users often save links by sending them to themselves on WhatsApp/Telegram, using browser bookmarks, saving platform-specific posts, or copying links into notes. Over time, saved links become scattered, unorganized, and hard to retrieve.

Users forget:

- Why they saved the link.
- What the link was about.
- Which platform it came from.
- Which category it belongs to.
- What keyword to search for later.

## 4. Core Solution

For every saved URL, Linkrai should:

1. Save the link immediately.
2. Extract metadata and available content.
3. Generate AI summary.
4. Suggest or create a category.
5. Generate tags and keywords.
6. Create embeddings for semantic search.
7. Allow search by exact words or meaning.
8. Keep the original URL accessible.

## 6. Main Use Cases

### Save Articles for Later

User finds an article and shares the URL to Linkrai. Linkrai summarizes it, tags it, and saves it under the right category.

### Save Social Media Posts

User saves LinkedIn, X/Twitter, Facebook, or other social links. Linkrai saves available metadata and analyzes accessible content.

### Save YouTube Videos

User saves a YouTube video. Linkrai extracts title, thumbnail, metadata, and later can analyze transcript if available.

### Save Business Ideas

A founder saves market research, competitor pages, pricing pages, and product ideas, then searches later using natural language.

### Search by Meaning

User searches for “the article about AI tools for clinics” and Linkrai returns relevant saved links even if the exact words are different.

## 9. Out of MVP Scope

- Browser extension.
- Team workspaces.
- Public collections.
- Collaboration.
- Full notes editor.
- PDF deep analysis.
- Advanced YouTube transcript analysis.
- Bookmark import.
- Knowledge graph.

## 10. Processing Transparency

Linkrai must always show the real processing state:

- Pending.
- Processing.
- Completed.
- Failed.
- Partial analysis.
- Metadata only.

The product must not claim full analysis when only metadata was available.
