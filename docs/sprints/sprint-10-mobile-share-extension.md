# Sprint 10 Tasks — Mobile Share Extension

## Goal

Allow users to share URLs from other apps into Linkrai.

## Tasks

- [ ] Implement Android share intent support.
- [ ] Implement iOS Share Extension.
- [ ] Extract shared URL from incoming shared text.
- [ ] Pass URL to Flutter app.
- [ ] Create saved item automatically or show quick confirmation.
- [ ] Show success feedback.
- [ ] Show failure feedback.
- [ ] Handle unauthenticated user by storing pending shared URL until login.

## Acceptance Criteria

- [ ] Android share to app works.
- [ ] iOS share to app works.
- [ ] Shared URL appears in library.
- [ ] Processing starts automatically.

## Scope Guard

Do not implement subscription purchases, backend billing webhooks, web dashboard, or future browser extension work in this sprint.
