# 13 — Linkrai Project Glossary

## Linkrai

The official product name.

## لينكراي

Arabic display name for Linkrai.

## Saved Item

A saved URL record in the user's library.

## AI Summary

A generated explanation of what a saved link is about.

## Category

A group that organizes saved links by topic.

## Tag

A keyword or label attached to a saved item.

## Semantic Search

Search by meaning using embeddings, not only exact keyword matching.

## Processing Status

The current state of a saved link analysis: pending, processing, completed, failed, partial, or metadata-only.

## Entitlement

The active paid access level for a user, usually managed through RevenueCat and enforced by the backend.

## Usage Counter

Monthly usage tracking for saved links, summaries, semantic searches, and future premium AI actions.
